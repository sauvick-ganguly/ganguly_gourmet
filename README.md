# Ganguly Gourmet

A full-stack MERN restaurant website inspired by Bengali fine dining.

## Features
- Responsive React frontend
- Table reservation system
- MongoDB Atlas database
- Express.js REST API
- CRUD operations for reservations

## Tech Stack
- React.js
- Express.js
- Node.js
- MongoDB Atlas
- Axios

## Installation

### Backend
cd backend
npm install
npm run dev

### Frontend
cd frontend
npm install
npm run dev
